Description

This app uses data sets from IMDB to perform complex SQL queries and trend analysis. These trends are visually represented using line graphs. The app performs six different queries using the IMDB data.

  1. Query 1: This query examines averages of movie ratings for a specific genre over a specified interval of time.
  2. Query 2: This query examines averages of movie ratings for a specific actor for each year of that actor's career.
  3. Query 3: This query examines averages of movie runtimes for each year within the specified time interval.
  4. Query 4: This query examines averages of movie ratings for each run time (in minutes) within the specified time interval.
  5. Query 5: This query finds the number of movies released for a specific genre within each year of the specified timer period.
  6. Query 6: This query finds the averages of TV show ratings for each season or individual episode.
